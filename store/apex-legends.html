    About Apex Legends Script

    Currently Undetected on Apex Legends (safe for use)

    Apex Legends Script: nyctum apex legends script, control recoil perfectly with ease, browse our features more in depth below. This product does not interact with Apex Legends in any way and is fully external from the game.

    General

    • menu that overlays apex legends
      • configurable in the keys section, default key is insert
    • mini menu
      • visualises the on/off state + the current weapon, attachment & barrel selected; control using arrow keys
    • recoil compensation for all full auto weapons
      • r301, flatline, r99, volt, alternator, devotion, havoc, re45, spitfire, lstar, rampage & car
    • recoil compensation for all sights
      • 1x reflex holosight, 1x hcog classic, 1x - 2x variable reflex holosight, 2x hcog brusier, 2x - 4x variable aog, 3x hcog ranger & 1x digital threat
    • recoil compensation for all barrels
      • common, rare, epic & legendary
    • recoil compensation for all magazines
      • common, rare, epic & legendary
    • customisable keybinds with configs
      • remembers your keybinds and options when exiting (hidden config file), also binds for exit/menu/reload cfg
    • navigation sections
      • recoil, aim assist, extra & settings sections on our user friendly menu

    Advanced

    • digital threat aim assist
      • smart humanized aim assist while you're using a digital threat sight/scope.
    • automatically detects sensitivity & fov and adjusts the script
      • grabs sensitivity & fov and adjusts accordingly (does not interact with the game)
    • auto detection for all full auto weapons
      • automatically detects the weapon that is being held and adjusts the script accordingly, works for all full auto weapons (does not interact with the game)
    • auto detection for all attachments
      • automatically detects the attachments on your weapon and adjusts the script accordingly, works for all attachments, including turbo charger and rampage charge (does not interact with the game
    • recoil pausing
      • turning the recoil on and off mid-spray is supported
    • re-aim control
      • if you begin hip firing in an un-ads state the script will know if you begin to ads and begin to control from that position without issues
    • cursor check
      • checks for cursor visibility so that the script only works when you are truly ready to use it
    • script only works in Apex Legends
      • script only works in the Apex Legends window to prevent the spray going off whilst you browse through your cat picture collection
    • sounds
      • beeps for when keybinds are hit

    Hipfire Options

    • hip-fire control
      • when the cursor isn't visible the script will control the spray when you begin aiming down sights, or un-ads (it will always control hipfire)
    • control hip-fire on ADS
      • will control hipfire when you have begun shooting and un-ads
    • never control hip fire
